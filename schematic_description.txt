4. VCO module
- https://www.allaboutcircuits.com/projects/diy-synth-series-vco/
- Input: potentiometer connected across VCC and 0V with varying output voltage connected to summing stage
- Input voltage changes pitch (1v = 1 octave)
- Knobs are Typically:
    - Frequency/Tune: Adjusts the pitch of the raw waveform (sawtooth, square, triangle, sine).
        - Second VCO circuit does this
    - Fine Tune: Provides subtle pitch adjustments for detuning against another oscillator for a thicker sound.
    - Waveform Select: A rotary switch or button that changes the shape of the wave, affecting the timbre
        - This will be in code using data tables that shift between sine wave, square wave, sawtooth, triangle, etc
